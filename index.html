Introductie

Ik ben Jamie Hart en ik ben student Built Envirnoment Stedenbouw op het HVA en ik volg de minor web design & development. Ik heb deze minor gekozen, omdat ik web design mij heel interessant en leuk leek en, omdat ik denk dat ik veel van wat ik hier leer later ook kan toepassen voor websites van projecten in de stedenbouw.

Dit is een blog van de ontwikkeling van mijn kennis van web design & development. Tijdens de minor zijn er bijna elke week gastsprekers geweest om te praten over een bepaald onderwerp die ons zou moeten helpen met de projecten en in deze blog is een overzicht van wat ik allemaal mee heb genomen hieruit en nu ermee kan.

Reflectie op de minor

In het begin van de minor, had ik twijfels over of ik het zou halen. De iedereen in de klas behalve ik had al ervaring met code, dus ik begon meteen met een grote achterstand. Ik had wel de basis van Html en CSS geoefend als voorbereiding, maar nog steeds had ik erg weinig kennis en met javascript had ik nog helemaal nog niet gewerkt.

Ondanks dat ik die achterstand had, heb ik gemerkt dat ik heel veel kan leren ergens over als ik er genoeg tijd in steek. Ik heb gemerkt dat als ik op zo een intensieve manier werk aan iets dat ik een korte tijd ongelofelijk veel vooruitgang kan maken. Van iemand die minimale kennis van coderen had in februari, ben ik gegaan naar iemand die nu vanaf niks een goed uitziende site kan coderen, met goeie code.

Ik hoop dat ik, wat ik heb geleerd kan gebruiken later. Ik ben in de korte tijd dat ik met code heb gewerkt erachter dat ik er ook zeker iets mee wil gaan DoelenList Voor projecten in de stedenbouw zijn er altijd informatie pagina's en ik hoop een manier te vinden om op een interactieve manier, met code informatie online aan te bieden aan toekomstige bewoners of andere belanghebbende. Met CSS en javascript moet dit wel mogelijk zijn.
